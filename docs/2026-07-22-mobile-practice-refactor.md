# 手机适配与跟练页重构记录

> 日期：2026-07-22  
> 范围：单页弹唱跟练工作区  
> 原则：保留上传、解析、播放、谱面同步、麦克风判定与专项练习逻辑，只重构练习界面与响应式交互。

## 记录一：跟练工作台一屏化

- 将原本上下堆叠的播放区、谱面和实时判定整合为 `practice-board`。
- 左侧 `lesson-column` 保留原始视频、左右手特写、波形、小节、和弦与六线谱。
- 右侧建立独立实时判定面板，集中显示判定状态、准确率、当前小节、目标动作、播放速度和专项练习入口。
- 工作区高度受 `100dvh` 约束，桌面端优先在一个视口内展示完整跟练内容。
- 进入最终跟弹页后隐藏品牌、课程摘要、麦克风与设置所在的整条顶部栏，将屏幕空间全部留给跟练内容。
- 缩短波形、和弦轨和六线谱的纵向尺寸；在低高度桌面端隐藏次要波形，保留小节、和弦和六线谱。
- 时间轴播放指针改为读取 CSS 轨道标签宽度，在不同屏幕宽度下保持对齐。

## 记录二：手机专用跟练模式

- 竖屏增加“老师画面 / 左手特写 / 右手特写”切换，不再将三块画面同时压缩。
- 切换按钮同步更新 `aria-pressed`，支持键盘与辅助技术识别当前画面。
- 手机六线谱缩放到当前屏宽，保留点击音符定位能力，取消上下滑动才能找到判定区的操作。
- 实时判定在竖屏收敛为底部紧凑面板，保留当前状态、准确率与专项练习入口。
- 手机横屏使用双栏沉浸式布局：左侧画面与谱面，右侧实时判定。
- 支持刘海屏和 Home Indicator 的顶部、底部及左右 `safe-area` 安全区。
- 手机全屏按钮会对当前选中的老师、左手或右手画面生效。
